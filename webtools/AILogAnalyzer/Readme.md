## AI Log Analyzer

AI Log Analyzer is a web tool that uses ArduPilot's log parser plus a configurable AI backend to explain flight logs in plain language. It supports a local OpenAI-compatible endpoint such as Ollama, Gemini with a Gemini API key, and ChatGPT/OpenAI with an OpenAI API key. Upload an ArduPilot binary log, ask questions like "why did altitude oscillate here?" or "show GPS health over time," and the analyzer will request relevant message data, summarize it, and render simple plots in the browser.
